# Task Management System

## Project Description

The Task Management System is a command-line application developed using Python. It helps users organize and manage their daily tasks efficiently.

The application allows users to:

- Add new tasks
- View all tasks
- Search tasks
- Update existing tasks
- Delete tasks
- Mark tasks as completed
- Filter tasks based on status and priority

All task information is stored permanently using a JSON file.

## File Description

### main.py
Starts the application and displays the main menu.

### task_manager.py
Contains all task management functions.

### file_handler.py
Handles reading from and writing to the JSON file.

### validation.py
Validates task title, priority, and due date.

### tasks.json
Stores all task details permanently.

## How to Run

1. Open the project in PyCharm.
2. Run `main.py`.
3. Choose an option from the menu.
4. Manage your tasks.
